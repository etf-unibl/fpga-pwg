Detailed Description

Architecture definition of the time counter.

Architecture implemented using methodology of regular structured sequential circuits with two registers for unix time and nanoseconds time. Unix time counter increments when nanoseconds coutner counts to 10^9 (1 second). Nanoseconds counter then resets. Asserting rst_i signal resets all counters to zeros. Asserting set_i signal sets value of unix time register to the time_i input value.
